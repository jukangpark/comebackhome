## 환경 변수 파일

### `ssh.env`
홈서버 관련 정보가 들어있습니다.

```

password=
```

### `.env`
Meshy API 키가 들어있습니다.

```
MESHY_KEY=
```

## 주의사항
- 위 파일들은 민감 정보(비밀번호, API 키)를 포함하므로 커밋하거나 외부에 노출하지 않습니다.
- 값이 필요할 때는 해당 파일에서 읽어서 사용합니다.
